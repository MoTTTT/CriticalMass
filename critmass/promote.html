Q Solutions Presents Critical Mass Volume 4

Introduction

Q Solutions is proud to present the fourth volume in the Critical Mass Open Source Software Distribution Series. This promotional brochure briefly outlines the contents of the Critical Mass Volume 4 CDROM, together with a description of the Critical Mass Project and Q Solutions. A price list with distribution and contact details will be found on the last page.

The Contents of Critical Mass Volume 4

Critical Mass Volume 4 is a CD containing the RedHat 6.0 GNU distribution of the Linux. operating system.

Critical Mass Volume 4 contains the following software and documentation:

  • Linux kernel 2.2: The Linux kernel provides a fully functional, robust, modern Unix like operating system. This multi user multi tasking environment provides the dependable software foundation required by the rest of the software
  • GUI Desktop environments: Desktop Environments provide features such as desktop icons, and drag-and-drop and cut-and-paste between applications running in the XFree86 Graphical User Interface. Both KDE (Kool Desktop Environment) and GNOME (GNU Network Object Model Environment) are included in this Linux distribution. To add to the choices provided, a number of user-friendly and easily configurable window managers are available. These provide functionality such as user menus, control of window behavior, and keyboard hotkeys.
  • Internet services: Linux gives native support for TCP/IP and all common Internet protocols such as TCP/IP, DNS (Domain Name Service), HTTP (web servers), SMTP (e-mail), FTP(File Transport Protocol), NNTP (news) and PPP/SLIP (dial-in accounts). Netscape Communicator is included on the CD allowing for both web browsing and the viewing and managing of e-mail accounts. The Apache web server which is very popular and highly regarded is also included on the distribution.
  • File and print services: These allow the system to access network resources and share files and printers with other users. Typical services include NFS (Network File System) to share files with other Unix based systems, and Samba that provides file and print services to Windows PCs. The Appletalk protocol is also available on the RedHat distribution.
  • Developer tools: Linux is a favourite of programmers and the RedHat distribution includes many developer tools. It includes utilities for managing and building programs from source code, debuggers such as gdb, compilers such as egcs and gcc, programming and scripting languages such as Python and TCL.
Updates to the Red Hat distribution as at the date of printing will be included on the CD. These include security updates and in some cases, new releases of programs in the distribution.

Other Volumes in the Critical Mass Series

  • Critical Mass Volume 1 contained the RedHat 5.1 release of GNU Linux.
  • Critical Mass Volume 2 contained the RedHat 5.2 release of GNU Linux.
  • Critical Mass Volume 3 contained the RedHat Powertools, a collection of additional applications for the Linux platform such as audio, graphics, administrative and programming tools, games, editors, and additional documentation.
  • Critical Mass Volume 4 contains the RedHat 6.0 release of GNU Linux.
Later volumes of the Critical Mass Series will include Caldera Open Linux and Debian (other well known Linux distributions), further releases of Red Hat PowerTools, source code to the GNU GPL software, and public Linux software archives.

The Critical Mass Project

As an Open Source distribution project the goal is to allow as many people as possible to benefit from the efforts of the Open Source development community. In order to achieve this the products are affordably priced.

All the software is distributed under the terms of the GNU General Public License. The source code for this software is available on the internet. If enough interest is shown, later volumes of Critical Mass will include source code CDs.

Target Market

The intended target market for Critical Mass Volume 4, and other Volumes in the Critical Mass Series are clients with the following attributes:
  • Clients familiar with the Linux operating system who want to upgrade their Linux distribution.
  • Clients who do not want to buy a users manual or support with their Linux software. Documentation is included on the CD.
  • Clients who want the latest release on CD before imported products become available on the shelves.
  • Clients who cannot afford highly priced software solutions.
  • Clients can not download their software from the internet.
Clients interested in printed documentation or official Red Hat support are encouraged to purchase official Red Hat products including these items.
